Alt Control

Pushy
Rocky

An experiment in Alt Ctrl games, we created two simple games for two players. We used Arduino boards for both projects, doing all the wiring/soldering/3D printing ourselves.

Pushy

Other Devs: Michael John

This game resembles an arm-wrestling competition, with players flexing to activate muscle sensors with a goal of turning all LEDs on the strip into their respective colors.

Rocky

Other Devs: Michael John, Sam Reha

This cooperative game evolved from Pushy and involves two people steering an LED ship on an LED strip. One person controls the left thrusters, while the other person controls the right. The goal is to land the ship on a target LED, shown in a different color. We created two versions of this game: one using the muscle sensors for input and one using potentiometers. Since we had access to a 3D printer, I was able to use my previous manufacturing experience to quickly model and print controllers for use with the potentiometers. Remember - SAFETY FIRST!
